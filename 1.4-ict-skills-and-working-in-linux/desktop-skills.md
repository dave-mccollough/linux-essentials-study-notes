## Userspace and privacy
- Each user has a `/home/` directory

- **Root**
    - User A
    - `/home/`

    - User B
    - `/home/`

    - User C
    - `/home/`

- Users may be able to view files, but they cannot edit other users files
- If you have root access, you have access to everything
